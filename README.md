# AutoCompetitor - 多Agent驱动的电商竞品情报自动化系统

## 项目解决的核心痛点
中小电商团队每天需手动监控数十个竞品店铺的价格、促销、新品及评价，琐碎、滞后且极易遗漏关键情报。本项目实现全自动多Agent协作情报系统，将人工2小时/天的监控压缩至5分钟。

## 核心推理流（长链推理 + 多Agent协作）
系统由4个Agent接力协作：

1. **感知Agent**：无头浏览器截取商品页面并提取可见文本
2. **多模态解析Agent**：调用视觉模型分析截图，提取价格、促销文案、新品标记、评价情感
3. **分析决策Agent**：加载历史数据，多步对比价格波动、评价情感突变、促销策略变更，生成三级预警标签（长链推理核心）
4. **报告生成Agent**：格式化为Markdown情报卡片并推送企业微信

## 技术栈
- Python
- Playwright（无头浏览器）
- OpenAI GPT-4o（多模态视觉模型）
- Git / GitHub
